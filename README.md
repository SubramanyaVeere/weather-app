🌤️ WEATHER APPLICATION
A dynamic weather application built with React that displays real-time weather data and animated background videos based on weather conditions.

## 🚀 Live Demo
https://your-vercel-link.vercel.app

## ✨ Features
- Real-time weather data using OpenWeather API
- Dynamic background videos (Rain, Clouds, Thunderstorm, Clear)
- Clean glassmorphism UI
- Responsive design (Mobile + Desktop)
- Error handling for invalid cities

## 🛠 Tech Stack
- React (Create React App)
- JavaScript (ES6+)
- CSS3
- OpenWeatherMap API
- Vercel (Deployment)

## 📦 Installation
Clone the repository:
git clone https://github.com/SubramanyaVeere/weather-app.git

Install dependencies: npm install

Start development server: npm start 

## 🔐 Environment Variables
Create a .env file in the root:
  REACT_APP_WEATHER_API_KEY=your_api_key_here

## 🌍 Deployment
Deployed using Vercel
