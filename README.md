# 🌤️ Weather App

----
A dynamic *Weather Application* built using *React* and the *OpenWeather API* that displays real-time weather information and a 5-day forecast.

---

## 🚀 Versions

🧩 *Weather V1 – CSS Version*  
A simple weather interface built using *HTML, CSS, and JavaScript*.

⚛️ *Weather V2 – React + Bootstrap Version*  
An improved version using *React components, **Bootstrap styling, and **API integration*.

---

## ✨ Features

🌍 Live weather data from OpenWeather API  
📅 5-Day weather forecast  
🌙 Dark / ☀️ Light theme support  
📱 Fully responsive design (mobile, tablet, desktop)  
🔍 City search functionality  
⚡ Fast React component rendering

---

## 🛠️ Tech Stack

- ⚛️ React
- 🎨 Bootstrap
- 🌐 OpenWeather API
- 💻 JavaScript
- 🧾 HTML / CSS
